\subsection{Detailed Description}
A utility class to add P\-X\-L pile up hits. 

This class is to be used by any P\-X\-L simulator (e.\-g. St\-Pxl\-Fast\-Sim\-Maker) to add pileup hits

This class conforms to the S\-T\-A\-R St\-Maker standards.

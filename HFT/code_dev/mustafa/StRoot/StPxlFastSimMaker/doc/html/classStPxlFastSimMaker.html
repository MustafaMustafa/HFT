Detailed Description

Class to simulate Pixel hits from Monte Carlo.

This class has the responsibility to create StPxlHit objects and store them in StPxlHitCollection which is later added to StEvent.

StPxlHit is a Gaussian smeared StMcPixelHit.

This class conforms to the STAR StMaker standards.

Int_t StPxlFastSimMaker::Make ( )
virtual

This is called every event, and contains all the activity of making StPxlHit objects.

Make() creates an StPxlHit object for every StMcPixelHit, and fills the hit StPxlHitCollection container. StPxlHitCollection container is passed to StEvent at the end.

Returns kStOk always.

\subsection{Detailed Description}
Class to simulate Pixel hits from Monte Carlo. 

This class has the responsibility to create St\-Pxl\-Hit objects and store them in St\-Pxl\-Hit\-Collection which is later added to St\-Event.

St\-Pxl\-Hit is a Gaussian smeared St\-Mc\-Pixel\-Hit.

This class conforms to the S\-T\-A\-R St\-Maker standards.
